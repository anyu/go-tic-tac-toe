# go-tic-tac-toe

Simple terminal tic-tac-toe written in Go.

## Usage

```shell
$ go run cmd/*.go

Let's play tic tac toe!

    0   1   2
  +---+---+---+
0 |   |   |   |
  +---+---+---+
1 |   |   |   |
  +---+---+---+
2 |   |   |   |
  +---+---+---+

Player 1's turn. Choose a spot (eg. '2,1' for top right corner):
```

```
    0   1   2
  +---+---+---+
0 | O | X | X |
  +---+---+---+
1 |   | X | O |
  +---+---+---+
2 | O | X |   |
  +---+---+---+

Winner is: Player 1 (X)

Want to play again? (y/n):
```
